# jvm-info
Prints information about the JVM.

## Install ##

From the root directory, run:
`mvn package`

An executable jar will be created in target/
Run this in the CLI:
`java -jar <NAME OF JAR>`

To help, create an alias in your ~/.bashrc or ~/.bash_profile
